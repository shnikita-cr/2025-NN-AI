# Построение моделей

## Задание

1.
    1. Загрузите датасет
    2. Разбейте код на смысловые ячейки
    3. Измените архитектуру модели для лучших результатов в задаче по датасету
    4. Обучите модель, сохраните веса в файл.
    5. Постройте график обучения (loss) и выведите метрику качества.
2. Проведите эксперимент с уменьшением датасета в 2 раза для обучения,
   обучите модель и сравните результаты, постройте графики для сравнения (два графика loss,
   столбчатая диаграмма для метрик)
