# Построение моделей CNN 

## Задание

0. Импортируйте новые пакеты. Скачайте [датасет]()
1. Переименуйте модель, измените архитектуру модели с использованием слоев свертки
2. Проведите обучение
3. Оцените полученные метрики с помощью tensorboard
4. Экспортируйте модель с помощью onnx
5. Откройте полученную экспортированную модель в [netron.app](https://netron.app/)
