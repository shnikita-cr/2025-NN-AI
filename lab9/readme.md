# Алгоритм **K ближайших соседей** для классификации

## Задание

0. Визуализируйте данные на двумерном графике с раскраской по классам
1. Постройте модель для определенного параметра K=N1
   1. Посчитайте **confusion matrix**
   2. Визуализируйте **confusion matrix** с помощью **seaborn**
   3. Посчитайте **F1 score**, **Accuracy**, **Precision** и выведите их
2. Повторите построение модели для K из отрезка [3, N2]
   1. Посчитайте **F1 score**, **Accuracy**, **Precision** и выведите их для каждого случая K
   2. Выведите их на 3 разных графика, по оси X отложите параметр K
3. Подберите параметр K автоматически с помощью GridSearchCV
   1. Оцените полученную модель, посчитайте **confusion matrix**
   2. Сравните метрики лучшей модели с полученными на шаге 1.3
