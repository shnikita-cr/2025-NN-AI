# Оценки качества

## Задание

0. Визуализируйте данные на двумерном графике с раскраской по классам
1. Проведите предобработку данных
2. Выберите модель классификации 
3. Обучите выбранную модель классификации
4. Оцените полученный результат
   1. Посчитайте `confusion_matrix` и визуализируйте ее с помощью `sns.heatmap` (тепловой карты)
   2. Выведите `classification_report` 
   3. Посчитайте **F1 score**, **Accuracy**, **Precision** и выведите их
   4. Выведите значение ROC AUC (`roc_auc_score`), визуализируйте ROC кривую (`RocCurveDisplay`)
